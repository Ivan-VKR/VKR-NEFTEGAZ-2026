# TankVKR

Программный расчётно-рекомендательный модуль для выпускной квалификационной работы по теме:

**«Разработка мероприятий по модернизации защитных покрытий резервуаров с разработкой программного алгоритма для расчёта основных элементов резервуарного парка»**

## О проекте

`TankVKR` — настольное приложение на Python с графическим интерфейсом `Tkinter`, предназначенное для предварительного инженерного расчёта типовых вертикальных стальных резервуаров для хранения нефти и нефтепродуктов.

Программа позволяет:
- выбрать типовой резервуар по номинальному объёму;
- выбрать нефтепродукт из встроенного справочника;
- задать город размещения резервуарного парка с автоматической подстановкой климатических параметров;
- выполнить расчёт основных геометрических, нагрузочных и проверочных показателей;
- получить рекомендации по конструктивному решению резервуара;
- получить рекомендации по защитным покрытиям;
- получить подбор листового проката и ориентировочную раскладку листов;
- получить рекомендуемый состав дополнительного оборудования резервуара;
- сохранить результаты в `.txt` и `.xlsx`;
- вывести поэтапные логи расчёта.

## Основные возможности

### 1. Ввод исходных данных
Пользователь задаёт:
- город размещения;
- номинальный объём резервуара;
- тип нефтепродукта;
- срок службы;
- температуру продукта;
- режим подогрева;
- категорию коррозионной среды (или автоматический подбор).

### 2. Автоматический подбор климатических параметров
По выбранному городу программа автоматически определяет:
- ветровой район;
- снеговой район;
- тип местности;
- минимальную температуру воздуха;
- максимальную температуру воздуха.

### 3. Расчёт резервуара
В расчёте определяются:
- диаметр резервуара;
- высота стенки;
- расчётная высота налива;
- площадь поперечного сечения;
- полезный объём;
- масса нефтепродукта;
- гидростатическое давление;
- ветровая нагрузка;
- снеговая нагрузка;
- ориентировочный собственный вес элементов;
- требуемая и принятая толщина поясов;
- напряжения в стенке;
- предварительная проверка прочности и устойчивости;
- необходимость ветрового кольца;
- параметры крыши и опорного контура.

### 4. Рекомендательные блоки
Программа формирует:
- рекомендацию по типу конструктивного решения:
  - без понтона;
  - с внутренним понтоном;
  - с плавающей крышей;
- рекомендации по защитным покрытиям:
  - для внутренней поверхности;
  - для наружной поверхности;
  - для паровоздушной зоны;
  - для зоны контакта с продуктом;
  - для крыши и верхних участков;
- рекомендации по дополнительному оборудованию резервуара;
- подбор нескольких вариантов листового проката.

## Нормативная база

В программе использованы расчётные зависимости, коэффициенты и инженерная логика на основе следующих нормативных документов:

- **ГОСТ 31385-2023** — Резервуары вертикальные цилиндрические стальные для нефти и нефтепродуктов. Общие технические условия;
- **СП 20.13330.2016** — Нагрузки и воздействия;
- **СП 16.13330.2017** — Стальные конструкции;
- **СП 22.13330.2016** — Основания зданий и сооружений;
- **ГОСТ 27751-2014** — Надёжность строительных конструкций и оснований. Основные положения;
- **СП 28.13330.2017** — Защита строительных конструкций от коррозии;
- **СП 155.13130.2014** — Склады нефти и нефтепродуктов. Требования пожарной безопасности;
- **ГОСТ 27772** — Прокат для строительных стальных конструкций;
- **ГОСТ 19903** — Сортамент листового горячекатаного проката.

> **Важно:** программа является учебно-демонстрационной разработкой для ВКР и не заменяет полноценный проектный расчёт, выполняемый специализированной организацией.

- `app.py`, `app.pyw` — точки входа в приложение;
- `gui.py` — графический интерфейс `Tkinter`;
- `calculations.py` — расчётная часть;
- `recommendations.py` — логика рекомендаций по конструкции, покрытиям, оборудованию и логам;
- `reporting.py` — сохранение результатов в `.txt` и `.xlsx`;

- `data_tanks.py` — типовые параметры резервуаров;
- `data_norms.py` — нормативные коэффициенты и инженерные настройки;
- `data_locations.py` — база городов и климатических параметров.

## Запуск проекта

### Запуск из Python

1. Установить Python 3.10+.
2. Установить зависимости:

```bash
pip install openpyxl
```

3. Запустить программу:

```bash
python app.py
```

или для Windows без консольного окна:

```bash
pythonw app.pyw
```

## Сборка в `.exe`

Для сборки готового исполняемого файла под Windows используется `PyInstaller`.

### Установка PyInstaller

```bash
py -m pip install -U pyinstaller
```

### Сборка

```bash
py -m PyInstaller --onefile --windowed --name TankVKR app.pyw
```

### Сборка с иконкой

```bash
py -m PyInstaller --onefile --windowed --name TankVKR --icon=tank_icon.ico app.pyw
```

После сборки готовый файл появится в папке `dist`.

## Логика работы программы

1. Пользователь выбирает город.
2. Программа автоматически подставляет климатические параметры.
3. Пользователь выбирает объём резервуара и нефтепродукт.
4. Программа подгружает типовой резервуар и свойства продукта.
5. Выполняется расчёт геометрических, нагрузочных и проверочных показателей.
6. Формируются рекомендации по:
   - конструктивному решению;
   - защитным покрытиям;
   - дополнительному оборудованию;
   - подбору листового проката.
7. Пользователь может сохранить результаты или открыть логи расчёта.

## Реализованные особенности

- стартовый лог-скрин перед запуском основного окна;
- открытие основного окна в развёрнутом виде;
- поиск по первым буквам в поле выбора города;
- автоматический подбор коррозионной категории;
- учёт режима подогрева продукта;
- подбор оборудования для обычных и подогреваемых резервуаров;
- отдельный режим «Логи» с поэтапным расчётом.

## Ограничения

- расчёты являются укрупнёнными и предназначены для учебной демонстрации;
- не учитываются все частные случаи проектирования конкретного резервуара;
- часть логики построена на инженерных допущениях, явно выделенных в коде;
- подбор оборудования и покрытий носит рекомендательный характер;
- данные по городам и климатическим параметрам используются как нормативно-справочная база для предварительного выбора.
