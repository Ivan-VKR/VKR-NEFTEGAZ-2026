Программа для ВКР: расчетно-рекомендательный модуль резервуарного парка.

Что изменено в этой версии:
1. Убран начальный выбор РВС / РВСП / РВСПК.
   Теперь пользователь выбирает только типоразмер резервуара по объему,
   а программа сама формирует рекомендацию:
   - без понтона;
   - с внутренним понтоном;
   - с плавающей крышей.

2. Для Windows добавлен файл app.pyw.
   Его можно запускать двойным щелчком или командой pythonw app.pyw.
   Тогда консольное окно не открывается.

3. Интерфейс увеличен:
   - расширена область исходных данных;
   - увеличены ширины полей;
   - кнопки расчета и сохранения выровнены по центру.

4. Усилен температурный блок.
   Вместо условного выбора "умеренные / холодные / жаркие" теперь вводятся:
   - минимальная температура наружного воздуха;
   - максимальная температура наружного воздуха;
   - температура продукта;
   - режим подогрева.

5. Температурный блок влияет на:
   - предварительную оценку испарительных потерь;
   - оценку риска конденсации;
   - выбор варианта: без понтона / с внутренним понтоном / с плавающей крышей;
   - рекомендации по наружным и внутренним покрытиям;
   - рекомендации по паровоздушной зоне и зоне контакта с продуктом.

6. Температурный блок НЕ используется как полноценный нормативный теплотехнический расчет.
   Это инженерно-рекомендательная часть демонстрационной модели для ВКР.

Запуск:
- обычный запуск: python app.py
- запуск без консольного окна в Windows: pythonw app.pyw

Зависимости:
- стандартная библиотека Python
- openpyxl
